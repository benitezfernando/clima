# Clima

Get weater with Fastify.

- Fastify
- Swagger documentation with Fastify plugin
- Tap for testing.

## For start in local

```
cp .env.example .env

npm install
npm run dev

# Run test with Tap
npm run test
```

Open `localhost:3000` for see the application.

Open `localhost:3000/documentation` for see Swagger API documentation.
